# Data Augmentation en Imágenes en Escala de Grises

Este repositorio presenta la implementación **manual** de técnicas de aumento de datos aplicadas a imágenes en escala de grises, utilizando MATLAB y documentadas mediante una presentación.

## Técnicas implementadas
- Conversión manual a escala de grises
- Volteado (flipping)
- Rotación con interpolación bilineal
- Traslación
- Escalamiento con interpolación bilineal
- Borrado aleatorio (Random Erase)
- Mezclado de regiones (CutMix)

## Tecnologías
- MATLAB (implementación manual)
